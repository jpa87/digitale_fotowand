Dieses Tool lässt sich ganz einfach selbst befüllen.

Fügen Sie Fotos neuer Kollegen im Verhältnis 1:1 in den Ordner /src/ hinzu.

Der Dateiname sollte so aufgebaut werden:

Kürzel_Vorname Name Name Name Name..._Fach1-Fach2-Fach3(_Sonderfunktionen im Volltext, wenn es welche gibt).jpg

Ausgeschiedene Kollegen löschen Sie, indem Sie deren Foto löschen.

Die Fächer werden abgekürzt. Sie können weitere in der Datei faecherkanon.php hinzufügen.
